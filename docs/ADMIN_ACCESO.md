# Acceso administrador — Política Digital

El administrador **no** entra por la misma pantalla que los alumnos.

- **Alumnos:** `/login` (o la URL base del sitio).
- **Administradores:** `/admin/login` (ruta exclusiva).

Ejemplo en producción: `https://[tu-dominio]/admin/login`.

No se muestra ningún enlace a administrador en la pantalla de login de alumnos.
